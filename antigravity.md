# Antigravity Project Instructions

## Technology Stack
* **Frontend:** React 19, TypeScript, Vite
* **Backend:** Express, Node.js, TypeScript
* **Runtime/Package Manager:** Bun
* **Database:** PostgreSQL (planned), Redis (planned)

## Agent Instructions
* **Context7 for Documentation:** ALWAYS use the Context7 MCP server (`resolve-library-id` and `query-docs`) to fetch up-to-date documentation whenever you need to integrate, setup, or debug a library, framework, SDK, or CLI tool. Do not rely solely on your internal training data for recent libraries.
* **Tools:** Prefer using specific tools like `view_file`, `write_to_file`, `replace_file_content` instead of terminal commands when applicable.
