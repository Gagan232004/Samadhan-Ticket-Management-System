Build a modern, enterprise-grade **AI Ticket Management Dashboard** for a SaaS application named **Samadhan**. The UI should be clean, minimal, responsive, and inspired by Linear, Jira, Zendesk, and GitHub with smooth animations, rounded cards, subtle shadows, and support for both light and dark themes.

### KPI Cards

Display the following metrics at the top of the dashboard:

* Total Tickets
* Open Tickets
* Tickets Resolved by AI
* AI Resolution Rate (%)
* Average Resolution Time
* SLA Compliance Rate

### AI Command Center (Hero Section)

Create a large AI-powered summary card at the top of the dashboard that displays insights such as:

* Total tickets analyzed today
* Tickets automatically resolved by AI
* AI Reply Acceptance Rate
* Predicted SLA breaches
* Predicted busiest support hour
* AI-generated recommendation for improving support efficiency

The AI Command Center should have a modern glassmorphism design with an attractive "✨ AI Insights" button.

### Analytics

Include interactive charts for:

* Ticket Creation Trend (Line Chart)
* Tickets Resolved vs AI Resolved (Area Chart)
* Ticket Status Distribution (Donut Chart)
* Tickets by Category (Bar Chart)
* Priority Distribution (Pie Chart)

### AI Performance Panel

Create a dedicated panel displaying:

* AI Replies Generated
* AI Replies Accepted
* Average AI Response Time
* AI Accuracy Score
* Customer Satisfaction for AI Responses

### AI vs Human Comparison

Display a comparison table with:

* Tickets Resolved
* Average First Response Time
* Average Resolution Time
* Customer Satisfaction Score
* Resolution Success Rate

### Customer Sentiment Analysis

Visualize customer sentiment using AI:

* 😊 Positive
* 😐 Neutral
* 😠 Negative

Include a sentiment trend chart over the last 7 days.

### SLA Risk Monitor

Display tickets that are close to breaching SLA with:

* Ticket ID
* Priority
* Remaining Time
* Assigned Agent
* Warning indicator

### Live Queue Monitor

Show real-time metrics:

* Waiting Tickets
* Active Agents
* AI Currently Processing
* Average Wait Time
* Oldest Pending Ticket

### Smart Recommendations

Create an AI Recommendations widget that suggests:

* Tickets that should be escalated
* Duplicate tickets that can be merged
* Best agent for reassignment based on expertise
* Knowledge Base articles that could resolve common issues
* Tickets likely to become high priority

### Recent Tickets

Display a searchable and sortable table with:

* Ticket ID
* Subject
* Customer
* Assigned Agent
* Category
* Priority
* Status
* Created Date
* Last Updated
* Actions (View, Edit, Delete)

### Agent Leaderboard

Display the top-performing support agents ranked by:

* Tickets Resolved
* Average Resolution Time
* Customer Satisfaction
* Productivity Score

### Notifications & Activity

Include:

* Recent Activity Timeline
* Latest Notifications
* AI-generated alerts
* SLA warnings
* New customer replies

### Technical Requirements

* React + TypeScript
* Tailwind CSS
* shadcn/ui
* Recharts for analytics
* Lucide React icons
* Fully responsive layout
* Beautiful loading skeletons
* Empty and error states
* Mock data for all widgets
* Clean component architecture
* Premium SaaS-quality UI with smooth animations and micro-interactions

The dashboard should look like a production-ready AI-powered helpdesk platform used by modern enterprises, with a strong emphasis on AI insights, analytics, and operational efficiency rather than a basic admin dashboard.
